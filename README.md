# Retail Sales Analysis and Prediction

## Overview

This project performs real-world retail sales analysis and future sales prediction using Python and Machine Learning.

## Features

- Statistical Analysis
- Data Visualization
- Correlation Analysis
- Linear Regression Model
- Future Sales Prediction
- Business Insights

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Outcome

The project identifies sales trends, analyzes retail performance, and predicts future sales using machine learning.
